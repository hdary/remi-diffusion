## Notebooks

Ce dossier accueillera les notebooks Jupyter du tutoriel.

Objectif:

- exécuter des analyses d'IRM de diffusion
- documenter des pipelines de traitement
- montrer des exemples reproductibles pour les utilisateurs du site

Tu peux ajouter ici:

- des notebooks `.ipynb`
- des scripts `.py` d'exécution ou d'automatisation

Bon format de nommage:

- `01-qc-dwi.ipynb`
- `02-preprocessing.py`

Si un notebook ou un script a besoin d'explications supplémentaires, crée un petit `README.md` dans le même sous-dossier.
